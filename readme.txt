首先这是一个通过爬取  网页版qq  URL：w.qq.com  
通过扫描二维码实现模拟登陆，然后在后台一直处于活跃状态，如果有好友发来消息，则爬取下
相应的信息，消息来源方，以及消息内容 ， 然后将其整合 post至 图灵API――一个聊天机器人
API并获取返回值，然后将消息发送给消息来源方的 聊天机器人。
/****************************************************************/
使用操作说明如下：
（1）javac Main.java  & java Main 

（2）扫描生成的 Login.jpg  （注意：如果长时间未扫描，二维码会失效，需要重新java Main）

（3）登陆以后 程序会自动抓取并打印 你拥有的联系人 以及 拥有的的群 
   （注意：  如果你的联系人很多很多，因为程序试用一个String 获取所有的联系人，所以如果你
联系人很多很多，栈内存会爆，所以鼓励采用 小号测试）

（4）登陆之后，可以用其他qq向他发送信息，它会自动回复。
